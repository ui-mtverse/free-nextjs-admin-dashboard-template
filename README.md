Helios Pro is a free Next.js admin dashboard template for developers building modern SaaS dashboards, ecommerce admin panels, CRM tools, analytics portals, internal tools, and production-ready web apps.

It includes a polished App Router setup, 100+ pages, responsive layouts, charts, data tables, ecommerce screens, auth pages, settings, profiles, and reusable UI components built with Next.js, React, TypeScript, and Tailwind CSS.

## Why Helios Pro?

Helios Pro is designed as a serious starting point for real products, not a tiny demo shell. It gives you the pages and UI patterns usually needed in dashboards from day one: navigation, analytics cards, charts, tables, ecommerce workflows, profile pages, settings, auth flows, error pages, theme controls, and responsive behavior.

Use it for:

- Free Next.js admin dashboard projects
- React admin dashboard templates
- SaaS dashboard starter kits
- Ecommerce admin panels
- CRM and sales dashboards
- Analytics and reporting portals
- Internal tools and operations consoles
- Startup MVP admin panels

## Features

- 100+ dashboard, app, ecommerce, auth, table, chart, and UI pages
- 10 dashboard variants for analytics, ecommerce, CRM, finance, SaaS, projects, marketing, AI, support, and logistics
- App Router architecture with route groups for dashboard and standalone pages
- TypeScript-first codebase with clean reusable components
- Tailwind CSS v4 styling and custom design tokens
- Light and dark mode support with next-themes
- Responsive desktop, tablet, and mobile layouts
- ApexCharts-powered dashboards and reporting screens
- jsvectormap geographic visualizations
- Advanced table patterns including sorting, filtering, pagination, selection, sticky headers, editable rows, drag and drop, and empty states
- Ecommerce pages for products, orders, customers, checkout, reviews, coupons, and inventory
- Auth screens for sign in, sign up, forgot password, reset password, two-step verification, lock screen, side layouts, and boxed layouts
- Settings, profile, team, roles, permissions, billing, notifications, connected apps, and activity pages
- Production-friendly project structure for fast customization

## Quick Start

Clone the repository, install dependencies, and start the local development server.

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.

You can also use Bun:

```bash
bun install
bun run dev
```

## Available Scripts

| Command | Description |
| --- | --- |
| `npm run dev` | Start the local Next.js development server |
| `npm run build` | Create a production build |
| `npm run start` | Start the production server after build |
| `npm run lint` | Run ESLint |
| `npm run db:generate` | Generate Prisma client |

## Project Structure

```text
src/
  app/
    (with-layout)/        Dashboard pages with sidebar and topbar shell
      dashboards/         10 dashboard variants
      apps/               Email, chat, calendar, kanban, notes, tasks, files, contacts, tickets, invoices
      ecommerce/          Product, order, customer, cart, checkout, review, coupon, inventory pages
      user-account/       Profile, settings, team, roles, permissions, billing, activity pages
      forms/              Form inputs, layouts, validation, upload, rich text, multi-step forms
      tables/             Data table examples and advanced table patterns
      charts/             Chart pages and reporting dashboards
      ui-components/      Component showcase pages
      pages/              Landing, pricing, FAQ, help, about, contact, changelog, roadmap
    (without-layout)/     Auth and error pages
    api/                  API routes
  components/             Layout, shared, chart, table, form, and UI components
  data/                   Mock datasets used by dashboard pages
  lib/                    Utilities, auth, database, formatting helpers
  css/                    Global styles and Tailwind theme files
public/                   Images, logos, icons, illustrations, and static assets
prisma/                   Prisma schema and database setup
```

## Customization Guide

- Update branding assets in `public/images/logo` and `public/favicon.svg`.
- Edit global theme values in `src/css/style.css`.
- Change dashboard navigation in the layout/sidebar components under `src/components/layouts`.
- Replace mock datasets in `src/data` with your API or database data.
- Connect auth and database flows through `src/lib` and `prisma`.
- Remove pages you do not need before shipping a production app.

## Deployment

Helios Pro can be deployed to any platform that supports Next.js applications.

```bash
npm run build
npm run start
```

Add database and auth environment variables only if you enable those integrations.

## FAQ

### Is Helios Pro free?

Yes. Helios Pro is the free MTVerse Next.js admin dashboard template for the GitHub community.

### Can I use it for commercial projects?

Yes, the included MIT license allows personal and commercial use.

### Does it include backend logic?

The project is frontend-first with Prisma and better-auth readiness. Pages work with demo/mock data and can be connected to your own APIs, database, or auth provider.

### Is this a simple landing page template?

No. It is a complete admin dashboard UI kit with dashboards, apps, ecommerce pages, tables, charts, auth pages, settings, and reusable UI components.

### Can I remove unused pages?

Yes. The App Router structure is modular, so you can remove folders you do not need.
